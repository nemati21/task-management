# Task Management


## Prerequisites

*  ***NodeJS = 12.13.0***

*  ***Database = MongoDB***
 

## Commands to run the service 

*  ***npm install -g nodemon***  


>  **Note:**  ***Run the below commands in the project directory***

*  ***npm install***

*  ***npm start***
  

## Commands to run the tests 

*  ***npm test***
